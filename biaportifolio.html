PROJETOS DO CURSO

  • ANIMAÇÃO

    Para criar um código de animação no Visual Studio Code, você precisará utilizar uma combinação de linguagens como HTML, CSS e JavaScript. Você pode começar criando um arquivo HTML e definindo os elementos que deseja animar. Em seguida, utilize o CSS para aplicar estilos e animações aos elementos. Por fim, use o JavaScript para controlar a interatividade e os eventos de animação.Existem muitos tutoriais e recursos online disponíveis para ajudá-lo a aprender a criar animações no Visual Studio Code.

  • BHASKARA

    Para fazer um código de Bhaskara no Visual Studio Code, você pode criar um novo arquivo com a extensão ".cpp" (para C++) ou ".py" (para Python) e implementar a fórmula de Bhaskara no código.

  • CALCULADORA

    Para fazer um código de calculadora no Visual Studio Code, você pode criar um novo arquivo com a extensão ".cpp" (para C++) ou ".py" (para Python) e implementar as operações matemáticas básicas (+, -, *, /) no código, permitindo que o usuário insira os números e escolha a operação desejada.

  • CARROS

    Para fazer o código de contagem de valores dos carros no Visual Studio Code, você pode criar uma variável contador e percorrer a lista de carros, incrementando o contador toda vez que encontrar um valor válido (por exemplo, um carro com preço acima de determinado limite).

  • CRÔNOMETRO

    Para fazer um código de cronômetro no Visual Studio Code, você pode utilizar a linguagem de programação de sua escolha (como JavaScript, Python, C++, etc.) e implementar a lógica do cronômetro utilizando funções de tempo disponíveis na linguagem. Por exemplo, em JavaScript, você pode usar a função setInterval para atualizar regularmente o tempo exibido e a função Date para obter o tempo atual.

  • DIA,TARDE E NOITE

    Para criar um código no Visual Studio Code que alterne entre "manhã", "tarde" e "noite" com base no horário atual, você pode usar a linguagem JavaScript e a função getHours() do objeto Date para obter a hora atual. Em seguida, você pode usar condicionais (if, else if, else) para definir as faixas de horário e exibir a saída correspondente.

  • LÂMPADA

    Para criar um código de lâmpada simples no Visual Studio Code, você pode usar JavaScript e HTML. Basta criar um arquivo HTML com botões para ligar e desligar a lâmpada, e um arquivo JavaScript para manipular a imagem da lâmpada conforme os botões são clicados. Execute o arquivo HTML em um navegador para ver a lâmpada interagindo.

  • SEMÁFORO

    Para criar um semáforo simples no Visual Studio Code, você pode usar HTML, CSS e JavaScript. Basta criar um arquivo HTML com a estrutura básica, definir o estilo do semáforo com CSS e usar JavaScript para alternar as cores dos elementos do semáforo em um intervalo de tempo. Execute o arquivo HTML em um navegador para ver o semáforo funcionando.
